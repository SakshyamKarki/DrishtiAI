# DrishtiAI

AI system to detect:
- Deepfake images
- AI-generated images

## Tech Stack
1. Django (Backend)

2. Frontend
    - Core stack
     > Framework : React
     > Build Tool : vite
     > Language : Javascript (ES Modules)
    
    - Authentication & State Management
     > Redux Toolkit

    - Routing
     > React Router DOM 
    
    - Forms & Validation
     > React Hook Form
     > Yup
     > @hookform/resolvers

    - API Communication
     > axios
    
    - File uploads
     > React dropzone
     
    - Notifications
     > React Toastify

    - Styling
     > Tailwind css


3. PyTorch (Models)

## Run Backend
> cd backend

> pip install -r requirements.txt

> python manage.py runserver

## Run Frontend
> cd frontend

> npm install

> npm start
